# Agentic Roadmap

`roadmap/features.json` is the source of truth for long-running feature work.

Each roadmap item maps to:

- one GitHub issue for planning and discussion
- one long-lived `feature/<id>` integration branch
- one draft PR from `feature/<id>` to `main`
- one or more implementation PRs targeting the feature branch, created by Copilot cloud agents, local Codex sessions, or human contributors

The feature branch is merged into `main` only after human review and approval.
